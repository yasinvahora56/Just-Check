# Day 3: Operators & Expressions

Welcome to Day 3! Operators are symbols that tell the Python interpreter to perform specific mathematical, relational, or logical calculations.

---

## Topics to Learn

### 1. Arithmetic Operators
Used to perform standard mathematical operations:

| Operator | Name | Description | Example |
| :---: | :--- | :--- | :--- |
| `+` | Addition | Adds two values. | `10 + 5` `15` |
| `-` | Subtraction | Subtracts second value from first. | `10 - 5` `5` |
| `*` | Multiplication | Multiplushes two values. | `10 * 5` `50` |
| `/` | Division | Divides first by second (always returns a float). | `10 / 4` `2.5` |
| `//` | Floor Division | Divides first by second, rounds down to nearest whole number. | `10 // 4` `2` |
| `%` | Modulus | Returns the remainder of division. | `10 % 3` `1` |
| `**` | Exponentiation | Left operand raised to the power of right. | `2 ** 3` `8` |

---

### 2. Comparison (Relational) Operators
Used to compare two values. They always return a boolean: `True` or `False`.

| Operator | Description | Example |
| :---: | :--- | :--- |
| `==` | Equal to | `5 == 5` `True` |
| `!=` | Not equal to | `5 != 3` `True` |
| `>` | Greater than | `5 > 8` `False` |
| `<` | Less than | `5 < 8` `True` |
| `>=` | Greater than or equal to | `5 >= 5` `True` |
| `<=` | Less than or equal to | `4 <= 3` `False` |

---

### 3. Logical Operators
Used to combine conditional statements:

* **`and`**: Returns `True` only if **both** conditions are true.
 * *Example:* `(5 > 3) and (10 > 5)` `True`
* **`or`**: Returns `True` if **at least one** condition is true.
 * *Example:* `(5 > 8) or (10 > 5)` `True`
* **`not`**: Reverses the logical state (turns `True` to `False` and vice versa).
 * *Example:* `not(5 > 3)` `False`
