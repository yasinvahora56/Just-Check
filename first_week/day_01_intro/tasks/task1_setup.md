# Task 1: Environment Setup Verification

Your first task is to verify that Python, VS Code, and Git are fully installed and configured on your local machine.

## Steps to Execute

1. Open your Command Prompt (cmd) or PowerShell.
2. Run the following command to check your Python installation:
  ```bash
  python --version
  ```
  **Expected Output:**
  `Python 3.x.x` (e.g., `Python 3.12.1` or higher)

3. Run the following command to verify Git is installed:
  ```bash
  git --version
  ```
  **Expected Output:**
  `git version 2.x.x`

4. Launch VS Code and open this folder (`python-internship-first-week`).
5. Open the built-in terminal in VS Code using the shortcut `Ctrl + \`` (backtick) or go to **Terminal -> New Terminal** on the top menu.
