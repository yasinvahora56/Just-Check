# Day 20: Project Development (Planning + Core Build)

Welcome to Day 20! You've made it to the Final Project! Over the next two days, you will build a complete, real-world application from scratch. 

Today is all about **planning** your project, setting up the foundation, and getting the core logic working.

---

## 1. Project Development Lifecycle

Building a real project is very different from solving isolated tasks. You need a structured approach:

1.  **Planning & Design**: What are we building? What features does it need?
2.  **Basic Structure**: Setting up files and creating empty functions (stubs).
3.  **Core Logic**: Making the functions actually work (input/output).
4.  **Data Storage**: Ensuring the data isn't lost when the app closes (File/JSON).
5.  **Menu System**: Creating the interactive loop for the user.

---

## 2. Project Options

Choose **ONE** of the following projects to build. (We will use the **Expense Tracker** as an example in today's tasks).

1.  **Expense Tracker**: A CLI app to log daily expenses, categorize them, and calculate totals.
2.  **Weather App**: An API-based app that lets users search for current weather in different cities and saves their search history.
3.  **Quiz Application**: A multiple-choice quiz game that loads questions from a JSON file and tracks the user's high score.
4.  **File Organizer**: A Python script that looks at a messy folder and automatically sorts files into subfolders (Images, Documents, Audio) based on their extensions.
5.  **Contact Manager (Advanced)**: Enhance your Week 2 project with file storage, search by phone number, email validation, and alphabetical sorting.
